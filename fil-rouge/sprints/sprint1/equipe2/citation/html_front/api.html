Accueil API

Documentation API

Bienvenue sur la documentation de l'API. Cette interface vous permet d'accéder librement aux données publiques du site : citations et auteurs. Toutes les réponses sont renvoyées au format JSON.

Liste des endpoints disponibles

  • /json/quotes
    Récupère la liste de toutes les ciations.
  • /json/quotes/1
    Récupère la citation avec l'id 1
  • /json/quotes/random
    Récupère une citation au hasard
  • /json/authors
    Récupère la liste des auteurs
  • /json/authors/1
    Récupère l'auteur avec l'id 1
  • /json/quotes
    Récupère la liste de toutes les ciations
  • /json/usage
    Aide à l'utilisation de cette API
